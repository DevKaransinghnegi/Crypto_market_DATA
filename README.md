Crypto Trend Bot

A web-based crypto trend prediction chatbot that analyzes long and short account data to provide market trend signals. The app also integrates a TradingView chart for any crypto pair, allowing users to visualize trends and predictions in real-time.\

Features

Predicts market trends as Bullish (Go Long) or Bearish (Go Short) based on uploaded account data.

Displays the confidence level of the prediction.

Integrated TradingView charts for any crypto pair (spot or futures).

Chatbot interface that responds to greetings, trend requests, and crypto pairs.

Fully responsive.

Tech Stack

Backend: FastAPI

Frontend: HTML, CSS, JavaScript

Machine Learning: scikit-learn (Random Forest Classifier)

Visualization: TradingView Widget

Data: CSV containing longAccount and shortAccount columns
